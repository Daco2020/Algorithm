### 문제 설명

자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.

예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

### 제한 사항

N의 범위 : 100,000,000 이하의 자연수

### 해결 방법

1\. n을 이터러블한 문자열로 바꾼다.

2\. 반복문을 통해 문자열을 리스트로 바꾼다.

3\. 리스트의 요소들을 정수로 바꾼다.

4\. 리스트의 요소들을 더해 합을 구한다.

```
def solution(n):
    answer = sum([int(i) for i in str(n)])
    return answer
```
