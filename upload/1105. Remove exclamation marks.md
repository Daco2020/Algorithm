Write function RemoveExclamationMarks which removes all exclamation marks from a given string.

<br><br>

## Solution:
```py
class RemoveExclamationMarks:
    def get(self, s):
        return s.replace("!", "")
        
remove_exclamation_marks = RemoveExclamationMarks().get
```
